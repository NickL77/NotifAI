# NotifAI GitHub Pages Site

Static GitHub Pages site for NotifAI privacy policy and SMS terms.

## Before Publishing

The contact email in `index.html` is `nicholas.j.liu@gmail.com`.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Push this repo to GitHub:

   ```sh
   git remote add origin git@github.com:YOUR_USERNAME/notifai-pages.git
   git add .
   git commit -m "Create GitHub Pages privacy policy"
   git push -u origin main
   ```

3. In GitHub, go to Settings > Pages.
4. Set the source to "Deploy from a branch".
5. Select the `main` branch and `/root` folder.
6. Use the published URL as your Privacy Policy URL in Twilio.
